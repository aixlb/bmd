# Changelog

## v1.0.0 · 2026-07-04

首个正式版。M0–M6 全部里程碑 + PDF V2 + 品牌视觉落齐，79 项前端（vitest）+ 28 项 Rust（cargo test）测试全绿。

### 新增

- **工作区全文搜索**：侧栏搜索框同时匹配文件名与内容（大小写不敏感；文件名命中优先、内容按命中次数排序；跳过隐藏项与 >5MB 文件）
- **Windows 无边框窗口**：标题栏自绘最小化/最大化/关闭，与 macOS 一体化标题栏观感对齐
- **启动动画**：logo 粒子汇聚 SplashScreen，`prefers-reduced-motion` 自动跳过，点击可跳过
- **品牌视觉**：全新应用图标（全平台尺寸）；编辑器/侧栏/AI 面板空状态插画；AI 回复头像
- **MiniMax 预设**：AI 面板新增 MiniMax-M2 内置端点
- **快捷键提示平台化**：Windows 显示 Ctrl+X，macOS 显示 ⌘ 符号
- 用户手册（MANUAL.md）

### 修复

- 暗色主题下模型/会话下拉弹层白底（WebView2 对半透明背景 select 的弹层回退）
- AI 流式回复 UTF-8 跨 chunk 断字
- SplashScreen 在窗口尺寸未就绪或 rAF 被节流（隐藏/后台窗口）时卡住不退场
- 主题切换后 Mermaid 图未按新主题重渲染

### 从 0.1.x 升级

直接安装覆盖即可，无破坏性变更：

- 设置、API Key（系统钥匙串）、会话与工作区索引全部沿用
- Windows 用户首次启动后窗口变为无边框自绘标题栏；旧版本残留的窗口状态会被自动纠正，无需手动处理
- 应用图标更换为新版小猫图标，任务栏固定的旧快捷方式图标可能需重新固定一次才会刷新
