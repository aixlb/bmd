## v1.0.5 · 2026-07-11

### 修复

- 修复安装 `1.0.4` 后“设置 → 关于”仍显示 `1.0.3`；关于页、插件 `app.version` 与 `minAppVersion` 校验现在自动跟随应用发布版本
- 新增版本一致性构建门禁，npm、Cargo、Tauri 与锁文件版本不一致时直接终止测试和生产构建，避免再次发布版本元数据互相矛盾的安装包

## v1.0.4 · 2026-07-11

### 修复

- 单击文件树轮播 Markdown/普通文本预览时，标签标题已经切换但编辑区仍停留在上一篇内容；AI 助手面板开启时同样可正确同步当前文档
- 正式打开、预览、新建、标签切换及拖入/系统打开的异步结果统一服从最后一次用户操作，较慢的旧文件读取不再抢回焦点
- 快速切换或清空工作区时，旧目录扫描不再覆盖新文件树或重新启动旧目录监听
- 文档切换后取消上一文档尚未执行的字数与大纲任务；预览槽位换文件时同步派发插件 `file-open` 并刷新标签溢出状态
- 工作区切换时完整停止旧 AI 请求和工具循环，清空旧 `@文件` 引用并隔离 RAG/会话恢复状态；损坏的 AI/布局本地配置安全回退
- 编辑过的临时预览在新建文件前同样提示保存；另存为已打开路径时拒绝制造双编辑器；删除含未保存修改的条目时明确警告；目录双击不再误触发重命名
- 表格、代码块与 AI 回复复制统一使用带兼容降级的剪贴板写入，权限拒绝时不再误报复制成功

## v1.0.3 · 2026-07-10

### 新增

- **文件树顶部工具栏**：打开文件夹后可直接点击小图标新建文件、新建文件夹，并在“类型分组 / 名称升序 / 名称降序”之间切换排序
- **表格整表复制**：表格悬浮操作条新增“复制”，可把当前表格按 Markdown 管道表复制到剪贴板
- **文本内容与编码识别**：冷门扩展和无扩展文本可按内容打开；支持 UTF-8/BOM、UTF-16 与常见本地编码并保持原编码写回，二进制内容自动拒绝
- **单标签预览模式**：在文件树或侧栏搜索结果中单击文件只复用一个斜体预览标签，反复切换不再堆满标签栏；双击文件或双击斜体标签可确认正式打开；预览中编辑后再切换/关闭文件会弹出“保存并固定打开 / 不保存 / 取消”的三选项确认
- **普通文本编辑**：`txt/csv/json/yaml/toml/log/ini/conf/xml/css/js/ts/py/sh/sql` 等常见文本文件可打开为纯文本编辑标签，支持保存、自动保存、侧栏搜索、快速打开与 AI `@文件`
- **图片查看**：`png/jpg/gif/webp/svg` 等图片可从文件树点击、拖入窗口或 `⌘O` 打开为只读预览标签（棋盘格衬底、外部修改自动刷新；预览协议按扩展名回 MIME）
- **AI 识图**：对话输入框粘贴截图或 🖼 选图（≤8MB）随消息发送，Anthropic/OpenAI 双协议图像消息映射；图片数据不落盘存档（仅留张数）
- **AI 工具调用（Agent，P1 只读）**：打开工作区后 AI 可自主列目录、读文档、全文搜索后再作答（默认开，设置可关）——Anthropic/OpenAI 双协议流式工具调用，最多 8 轮；路径经 Rust 侧 canonicalize 强制锁在工作区内（防 `../` 与符号链接逃逸）；回复中可展开每步调用轨迹；模型不支持工具时自动降级纯对话
- **列表快捷键**：无序 `⌘L`、有序 `⌘⌥O`、任务 `⌘⌥X`（Windows 为 `Ctrl+L` / `Ctrl+Alt+O` / `Ctrl+Alt+X`）——已是该列表再按一次取消；跨类型切换自动换前缀不叠加；有序列表自动递增编号；空行触发直接放前缀开写

### 变更

- **文件操作事务化**：新建、重命名、删除统一经过文件操作层；目录重命名会同步更新内部全部标签，扩展名变化会重算编辑器类型，失败原因在应用内提示
- **可取消全文搜索**：输入新关键词会取消旧扫描；侧栏与 AI 搜索任务彼此隔离，Rust 文件遍历移入阻塞线程池
- **标题快捷键升级为 `⌘1–6`**（Windows `Ctrl+1–6`；原 `⌘⌥1–6` 保留为兼容别名，`⌘⌥0` 回正文不变）
- **标签直达从 `⌘1–9` 改为 `Alt+1–9`**（为标题切换让位；`Alt+9` 恒为最后一个标签，`Ctrl+Tab` 轮换不变）

### 修复

- 保存进行中继续输入时，旧保存完成会错误清除未保存标记，极端情况下导致后续自动保存跳过新内容
- 快速连续单击多个文件时，较早的异步读取可能覆盖最后一次点击，或产生多个临时预览标签
- 预览转正式、单文件启动或打开文件后立即退出时，会话防抖尚未落盘；现在窗口销毁前强制刷新
- 重命名文件夹后内部已打开标签仍指向旧路径，以及 HTML/图片重命名后预览地址失效
- 应用卸载时未完整释放文件拖放、系统打开、文件监听、插件事件桥和延迟计时器
- 连续快速切换工作区时，AI 会话迁移可能按过期的当前根加载或写错存档；窗口销毁前现在也会等待迁移并显式执行插件卸载
- Windows 覆盖保存已有文件时，标准重命名可能无法完成替换；现在使用系统原子替换并要求写穿落盘
- macOS 无开发者证书的构建现在也会完整执行 ad-hoc bundle 签名，应用资源可通过严格完整性校验
- 从系统打开上次工作区外的单个文件时，不再先恢复旧工作区文件夹；窗口会进入无工作区的单文件状态
- 窗口关闭时若临时预览标签仍有未保存修改，会先拦截关闭并提示保存/不保存/取消
- macOS 冷启动双击 `.md` 打开 bmd 后文件未加载：Apple Event 的 `Opened` 事件早于前端监听器注册、`open-file` 事件丢失（Rust 侧新增路径缓冲，前端就绪后经 `initial_files` 取走；app 已在运行时双击不受此影响）

## v1.0.1 · 2026-07-05

### 新增

- **标签右键菜单**：右键文件标签可关闭当前文件、关闭左侧/右侧文件、关闭其他文件（脏文件逐个确认，不适用的菜单项自动隐藏）
- **标签翻阅按钮**：打开文件过多溢出时显示 «/» 按钮，可向左/向右翻阅标签；激活标签自动滚入可视区；新建（+）按钮固定不随标签滚动
- **侧边栏展开/隐藏**：活动栏新增侧边栏开关按钮（对应快捷键 ⌘\ / Ctrl+\），显示状态持久化、重启后恢复
- **AI 多会话标签栏与并行提问**：会话入口从隐藏下拉改为始终可见的标签栏（＋ 新建、点击切换、× 关闭）；busy 状态按会话隔离，一个会话等待回复时其他会话仍可提问，进行中的会话标签显示呼吸圆点，关闭会自动取消其请求

### 修复

- AI 面板流式回复强制滚底：用户往上滚动即暂停自动滚底，滚回底部附近自动恢复；发送新消息或切换会话时跳回底部
- AI 面板反复开关会用磁盘快照覆盖进行中的会话（历史恢复现在只执行一次）
- 「停止」后旧请求的迟到增量仍会追加到消息、且其收尾会误清新请求的状态（按请求 id 校验清理，迟到增量直接丢弃）
- 快捷指令的 `{doc}` 占位符未被替换，原样发给了模型
- Windows 反斜杠路径下，文件树右键「新建文件/文件夹」解析父目录错误
- 启动空闲预热调用 `requestIdleCallback` 未绑定 this，会抛 Illegal invocation
- KaTeX 公式错误信息未转义直接拼入 HTML（注入面）；mermaid 并发渲染失败时可能清理错残留节点
- RAG 语义检索在嵌入服务返回空数据时整应用崩溃（现自动回退 BM25 词法检索）
- AI 请求新增 15s 连接超时；请求结束统一清理取消标记，不再缓慢泄漏
- 聊天存档改为原子写，崩溃不再截断历史；文档保存的临时文件名不再在 `a.md`/`a.markdown` 间碰撞
- 新建/重命名条目名称校验拦截盘符相对路径（如 `C:x.md`）与 `..`
- AI 面板每次开关泄漏一个全局事件监听器
- 缩进表格就地编辑写回区间错位可损坏文档；区间对不上时放弃写回兜底
- 相邻行编辑使块级元素（公式/表格等）边界外扩时产生重叠装饰
- 切换工作区后旧工作区聊天会写进新工作区存档；现按旧根落盘、按新根恢复，进行中请求自动取消
- AI 流式回复停滞时「停止」按钮无法生效；取消现在即时中断连接（连接阶段亦可取消）
- 聊天存档/RAG 索引文件名哈希改用 sha256（跨 Rust 版本稳定），旧命名存档自动迁移
- 新增 stop_watch 命令；文档保存的临时文件名与 watcher 忽略表保持一致
- 超长代码块/引用块在光标移动时全量重建行装饰的性能问题（改为只处理可见区间）
- 行内公式按 Pandoc 邻接约束判定，`价格$5 和 $10` 不再误判为公式
- 在加粗文本内使用斜体会剥掉加粗标记（现正确嵌套为 `***`）
- RAG 检索来源按会话隔离，并行提问不再互相覆盖
- 带点号的文件夹名（如 `my.notes`）拖入窗口无法作为工作区打开
- 表格 Tab 新建行的焦点恢复按编辑器实例隔离；右键输入弹窗并发不再悬挂；侧栏/AI 面板拖宽兼容 pointercancel；AI 流式渲染不再污染 markdown 缓存

### 安全

- **CSP 去掉 `unsafe-eval`**：插件装载机制由 `new Function` 改为 Blob ESM 动态导入（插件源码格式不变，详见 PLUGINS.md 变更记录 v1.1），`script-src` 收紧为 `'self' blob:`——HTML 注入类漏洞无法再升级为任意代码执行
- **HTML 预览 iframe 收紧沙箱**：移除 `allow-same-origin`，预览页脚本被隔离在匿名源，无法触达应用上下文与 Tauri IPC

## v1.0.0 · 2026-07-04

首个正式版。M0–M6 全部里程碑 + PDF V2 + 品牌视觉落齐，79 项前端（vitest）+ 28 项 Rust（cargo test）测试全绿。

### 新增

- **工作区全文搜索**：侧栏搜索框同时匹配文件名与内容（大小写不敏感；文件名命中优先、内容按命中次数排序；跳过隐藏项与 >5MB 文件）
- **Windows 无边框窗口**：标题栏自绘最小化/最大化/关闭，与 macOS 一体化标题栏观感对齐
- **启动动画**：logo 粒子汇聚 SplashScreen，`prefers-reduced-motion` 自动跳过，点击可跳过
- **品牌视觉**：全新应用图标（全平台尺寸）；编辑器/侧栏/AI 面板空状态插画；AI 回复头像
- **MiniMax 预设**：AI 面板新增 MiniMax-M2 内置端点
- **快捷键提示平台化**：Windows 显示 Ctrl+X，macOS 显示 ⌘ 符号
- 用户手册（MANUAL.md）

### 修复

- 暗色主题下模型/会话下拉弹层白底（WebView2 对半透明背景 select 的弹层回退）
- AI 流式回复 UTF-8 跨 chunk 断字
- SplashScreen 在窗口尺寸未就绪或 rAF 被节流（隐藏/后台窗口）时卡住不退场
- 主题切换后 Mermaid 图未按新主题重渲染

### 从 0.1.x 升级

直接安装覆盖即可，无破坏性变更：

- 设置、API Key（系统钥匙串）、会话与工作区索引全部沿用
- Windows 用户首次启动后窗口变为无边框自绘标题栏；旧版本残留的窗口状态会被自动纠正，无需手动处理
- 应用图标更换为新版小猫图标，任务栏固定的旧快捷方式图标可能需重新固定一次才会刷新
