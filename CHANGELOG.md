# Changelog

## v1.0.2 · 2026-07-06

### 新增

- **设置 →「关于」**：应用与版本信息、GitHub 仓库/用户手册/更新日志入口、作者 @玩AI的小笼包 联系链接（外链经系统浏览器打开）
- **Windows 绿色便携版**：发布产物新增 `bmd_x.y.z_x64-portable.zip`（解压即用、免安装，此后每个版本默认附带）；手册补充各系统「设为 .md 默认打开方式」指引
- **HTML 只读预览**：`.html/.htm` 文件可经文件树点击、拖拽入窗、`⌘O` 或系统「打开方式」打开为预览标签——经 `bmdpreview://` 自定义协议在 iframe 中渲染（相对路径的图片/CSS/JS 均可加载，外部修改自动刷新），仅预览、不支持编辑与保存

### 修复

- 生产包里启动粒子动画从未播放：splash logo 经 `blob:` URL 装载被 CSP `img-src` 拦截后走静默跳过兜底（v1.0.0 起存在；改用 `data:` URL 装载）
- 启动时控制台一条 `ipc.localhost` 被 CSP 拦截的报错：`connect-src` 未显式声明、回落 `default-src 'self'` 误拦 Tauri IPC 探测（窗控功能不受影响，纯噪音；现显式放行）

## v1.0.1 · 2026-07-05

### 新增

- **标签右键菜单**：右键文件标签可关闭当前文件、关闭左侧/右侧文件、关闭其他文件（脏文件逐个确认，不适用的菜单项自动隐藏）
- **标签翻阅按钮**：打开文件过多溢出时显示 «/» 按钮，可向左/向右翻阅标签；激活标签自动滚入可视区；新建（+）按钮固定不随标签滚动
- **侧边栏展开/隐藏**：活动栏新增侧边栏开关按钮（对应快捷键 ⌘\ / Ctrl+\），显示状态持久化、重启后恢复
- **AI 多会话标签栏与并行提问**：会话入口从隐藏下拉改为始终可见的标签栏（＋ 新建、点击切换、× 关闭）；busy 状态按会话隔离，一个会话等待回复时其他会话仍可提问，进行中的会话标签显示呼吸圆点，关闭会自动取消其请求

### 修复

- AI 面板流式回复强制滚底：用户往上滚动即暂停自动滚底，滚回底部附近自动恢复；发送新消息或切换会话时跳回底部
- AI 面板反复开关会用磁盘快照覆盖进行中的会话（历史恢复现在只执行一次）
- 「停止」后旧请求的迟到增量仍会追加到消息、且其收尾会误清新请求的状态（按请求 id 校验清理，迟到增量直接丢弃）
- 快捷指令的 `{doc}` 占位符未被替换，原样发给了模型
- Windows 反斜杠路径下，文件树右键「新建文件/文件夹」解析父目录错误
- 启动空闲预热调用 `requestIdleCallback` 未绑定 this，会抛 Illegal invocation
- KaTeX 公式错误信息未转义直接拼入 HTML（注入面）；mermaid 并发渲染失败时可能清理错残留节点
- RAG 语义检索在嵌入服务返回空数据时整应用崩溃（现自动回退 BM25 词法检索）
- AI 请求新增 15s 连接超时；请求结束统一清理取消标记，不再缓慢泄漏
- 聊天存档改为原子写，崩溃不再截断历史；文档保存的临时文件名不再在 `a.md`/`a.markdown` 间碰撞
- 新建/重命名条目名称校验拦截盘符相对路径（如 `C:x.md`）与 `..`
- AI 面板每次开关泄漏一个全局事件监听器
- 缩进表格就地编辑写回区间错位可损坏文档；区间对不上时放弃写回兜底
- 相邻行编辑使块级元素（公式/表格等）边界外扩时产生重叠装饰
- 切换工作区后旧工作区聊天会写进新工作区存档；现按旧根落盘、按新根恢复，进行中请求自动取消
- AI 流式回复停滞时「停止」按钮无法生效；取消现在即时中断连接（连接阶段亦可取消）
- 聊天存档/RAG 索引文件名哈希改用 sha256（跨 Rust 版本稳定），旧命名存档自动迁移
- 新增 stop_watch 命令；文档保存的临时文件名与 watcher 忽略表保持一致
- 超长代码块/引用块在光标移动时全量重建行装饰的性能问题（改为只处理可见区间）
- 行内公式按 Pandoc 邻接约束判定，`价格$5 和 $10` 不再误判为公式
- 在加粗文本内使用斜体会剥掉加粗标记（现正确嵌套为 `***`）
- RAG 检索来源按会话隔离，并行提问不再互相覆盖
- 带点号的文件夹名（如 `my.notes`）拖入窗口无法作为工作区打开
- 表格 Tab 新建行的焦点恢复按编辑器实例隔离；右键输入弹窗并发不再悬挂；侧栏/AI 面板拖宽兼容 pointercancel；AI 流式渲染不再污染 markdown 缓存

### 安全

- **CSP 去掉 `unsafe-eval`**：插件装载机制由 `new Function` 改为 Blob ESM 动态导入（插件源码格式不变，详见 PLUGINS.md 变更记录 v1.1），`script-src` 收紧为 `'self' blob:`——HTML 注入类漏洞无法再升级为任意代码执行
- **HTML 预览 iframe 收紧沙箱**：移除 `allow-same-origin`，预览页脚本被隔离在匿名源，无法触达应用上下文与 Tauri IPC

## v1.0.0 · 2026-07-04

首个正式版。M0–M6 全部里程碑 + PDF V2 + 品牌视觉落齐，79 项前端（vitest）+ 28 项 Rust（cargo test）测试全绿。

### 新增

- **工作区全文搜索**：侧栏搜索框同时匹配文件名与内容（大小写不敏感；文件名命中优先、内容按命中次数排序；跳过隐藏项与 >5MB 文件）
- **Windows 无边框窗口**：标题栏自绘最小化/最大化/关闭，与 macOS 一体化标题栏观感对齐
- **启动动画**：logo 粒子汇聚 SplashScreen，`prefers-reduced-motion` 自动跳过，点击可跳过
- **品牌视觉**：全新应用图标（全平台尺寸）；编辑器/侧栏/AI 面板空状态插画；AI 回复头像
- **MiniMax 预设**：AI 面板新增 MiniMax-M2 内置端点
- **快捷键提示平台化**：Windows 显示 Ctrl+X，macOS 显示 ⌘ 符号
- 用户手册（MANUAL.md）

### 修复

- 暗色主题下模型/会话下拉弹层白底（WebView2 对半透明背景 select 的弹层回退）
- AI 流式回复 UTF-8 跨 chunk 断字
- SplashScreen 在窗口尺寸未就绪或 rAF 被节流（隐藏/后台窗口）时卡住不退场
- 主题切换后 Mermaid 图未按新主题重渲染

### 从 0.1.x 升级

直接安装覆盖即可，无破坏性变更：

- 设置、API Key（系统钥匙串）、会话与工作区索引全部沿用
- Windows 用户首次启动后窗口变为无边框自绘标题栏；旧版本残留的窗口状态会被自动纠正，无需手动处理
- 应用图标更换为新版小猫图标，任务栏固定的旧快捷方式图标可能需重新固定一次才会刷新
