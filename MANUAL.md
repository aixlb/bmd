# bmd 用户手册

bmd（包 markdown）是一款所见即所得（WYSIWYG）Markdown 编辑器。本手册随仓库与版本同步更新，当前版本见 设置 →「关于」。

> 快捷键统一用 macOS 符号书写：`⌘` = Windows `Ctrl`，`⇧` = `Shift`，`⌥` = `Alt`。
> 应用内所有提示会按你的平台自动显示正确写法。

## 1. 安装

从 [GitHub Releases](https://github.com/aixlb/bmd/releases) 下载对应平台安装包：

| 平台 | 包 | 说明 |
|---|---|---|
| Windows 10/11 | `bmd_x.y.z_x64-setup.exe`（NSIS） | 首次安装自动配置 WebView2，注册 `.md` 文件关联 |
| Windows 绿色便携版 | `bmd_x.y.z_x64-portable.zip` | 解压即用、免安装；需系统已有 WebView2；不注册文件关联 |
| macOS（Apple Silicon） | `bmd_x.y.z_aarch64.dmg` | 需 macOS 11+ |
| macOS（Intel） | `bmd_x.y.z_x64.dmg` | 需 macOS 11+ |

**把 bmd 设为 .md 的默认打开方式**（安装版已注册关联，「打开方式」列表里能看到 bmd；设为默认需要你亲手点一次，系统不允许应用自行抢默认）：

- **Windows**：双击任意 `.md` → 弹出「打开方式」时选 **bmd** 并勾选「**始终**」；或 设置 → 应用 → 默认应用 → 按文件类型 → `.md` → 选 bmd
- **macOS**：任选一个 `.md` 右键 → 显示简介 → 「打开方式」选 bmd → 点「**全部更改…**」

## 2. 快速上手

- **打开文件** `⌘O`，**新建** `⌘N`，**保存** `⌘S`（另存为 `⌘⇧S`）
- **打开文件夹（工作区）** `⌘⇧O`——左侧出现文件树，自动索引供全文搜索与 AI 检索
- 直接开始打字：bmd 是即时渲染——光标进入元素时展开 Markdown 源码标记，离开立即渲染回排版效果。文档本体始终是纯文本 `.md`，编辑只改你动过的字符，不会整篇重排。

## 3. 编辑

- **格式快捷键**：`⌘B` 加粗 · `⌘I` 斜体 · `⌘E` 行内代码 · `⌘K` 插入链接 · `⌘⇧X` 删除线；标题 `⌘1–6`（`⌘⌥0` 回正文，`⌘⌥1–6` 兼容可用）；列表：无序 `⌘L` · 有序 `⌘⌥O` · 任务 `⌘⌥X`（再按取消，跨类型直接换型）；插入代码块/引用/公式块/表格 `⌘⇧K`/`⌘⇧Q`/`⌘⇧M`/`⌘⇧T`
- **源码模式** `⌘/`：整篇切换到纯源码视图，再按切回
- **Slash 命令**：行首输入 `/` 唤出插入菜单（标题、表格、代码块、公式、Mermaid…）
- **浮动工具条**：选中文本后浮现，一键加粗/斜体/链接等
- **查找替换** `⌘F`
- **表格**：单元格内直接就地编辑，Tab 跳格；鼠标移到表格上方会出现操作条，可增删行列、复制整个表格为 Markdown，或切回源码编辑
- **数学公式**：`$...$` 行内、`$$...$$` 块级，KaTeX 渲染
- **Mermaid**：\`\`\`mermaid 代码块实时出图，随主题重渲染
- **图片**：截图/图片直接粘贴，自动保存进文档同级 `assets/` 并写入相对路径
- **外部变更**：文件被其他程序改动时顶部出横幅，可一键加载磁盘版本

## 4. 工作区

- **文件树**：拖拽调宽，顶部按钮可新建文件、新建文件夹、切换排序，右键可新建/重命名/删除；单击文件只在右侧复用一个斜体预览标签，反复找文件不会堆满标签栏；双击文件或双击斜体标签会固定为正式打开
- **普通文本**：`txt/csv/json/yaml/toml/log/ini/conf/xml/css/js/ts/py/sh/sql`、`LICENSE`、`.gitignore` 等常见文本会以纯文本编辑器打开；冷门扩展也会按内容识别。支持 UTF-8/BOM、UTF-16 与常见本地编码并保持原编码写回，疑似二进制文件不会误进文本编辑器
- **全文搜索**：侧栏顶部搜索框，同时匹配文件名与可读取文本内容（文件名命中优先）；输入变化会取消旧搜索，跳过未知隐藏项、二进制与超大文件
- **快速打开** `⌘P`：模糊匹配跳转工作区内的可打开文件
- **大纲**：侧栏「大纲」页签，点击标题跳转
- **多标签**：`Ctrl+Tab` 轮换、`⌥1–9` 直达（`⌥9` 恒为最后一个）、`⌘W` 关闭；未保存的标签带圆点标记。预览标签中编辑后再切换、关闭文件或退出应用，会先选择保存、不保存或取消；确认保存后才转为正式标签
- **自动保存**与**会话恢复**：保存期间继续输入也不会被误判为已保存；退出前会等待在途保存并立即写入会话。重启后回到上次打开的正式标签；若从系统直接打开一个不在上次工作区内的单独文件，则进入无文件夹的单文件窗口，不再展示旧工作区
- **HTML 只读预览**：`.html/.htm` 从文件树点击、拖入窗口或 `⌘O` 打开为预览标签——相对路径的图片/CSS/JS 正常加载，文件被外部修改自动刷新；仅预览、不可编辑
- **图片查看**：`png/jpg/gif/webp/svg` 等图片同样可点击/拖入/`⌘O` 打开为只读预览标签（棋盘格衬底，外部修改自动刷新）

## 5. 导出

设置（`⌘,`）→「导出当前文档」：

- **导出 HTML**：单文件、内嵌样式，随主题明暗
- **导出 PDF**：原生静默分页导出，无打印对话框——macOS 走 WKWebView 打印管线（需 macOS 11+），Windows 走 WebView2 `PrintToPdf`；文档内相对路径图片正确嵌入；原生路径不可用时自动回退系统打印对话框

## 6. AI 写作副驾

**唤醒/收起**：`⌘J`，或点面板右上角 ×。

**配置模型**：面板顶部 ⚙ →选预设、填 API Key、保存。内置预设：

| 预设 | 默认模型 |
|---|---|
| Claude | claude-sonnet-5 |
| OpenAI | gpt-4o |
| DeepSeek | deepseek-chat |
| Kimi | moonshot-v1-32k |
| MiniMax | MiniMax-M2 |
| 通义千问 | qwen-plus |
| 智谱 GLM | glm-4-plus |
| Ollama 本地 | qwen2.5（免 Key，本机 11434 端口） |

模型名可改；预设不够用时「＋ 自定义模型端点」可接任何 OpenAI 兼容接口。**API Key 保存在系统钥匙串**（macOS 钥匙串 / Windows 凭据管理器），不写配置文件，无遥测。

**使用**：

- 输入框 Enter 发送、Shift+Enter 换行；当前文档与选区自动作为上下文携带
- **快捷操作**：续写 / 润色 / 翻译 / 总结 / 生成大纲，一键套用到当前上下文
- **上下文控制**：「选区」只带选中段落；「@文件」附加工作区其他 Markdown/文本文件；「RAG」开启工作区检索，AI 可引用相关章节作答
- **工具调用（Agent）**：打开工作区后 AI 可自主查阅文档——列目录、按需读取文件、全文搜索（**只读**，且被强制限制在工作区内），回复里可展开每一步调用轨迹；设置 → AI 助手可关闭。模型不支持工具时自动降级为纯对话
- **识图**：输入框直接**粘贴截图**，或点 🖼 选择本地图片（≤8MB），随消息发给模型——需模型支持视觉输入（如 moonshot vision / GPT-4o / Claude）；图片不写入会话存档，历史里仅显示张数
- **应用改动**：回复下方「插入」到光标处，或「替换选区」——先弹 diff 预览，确认后才落盘
- **多会话**：面板顶部会话标签栏，＋ 新建、点击切换、× 关闭；不同会话可**并行提问**，互不阻塞——标签上的呼吸圆点表示该会话正在等待回复

## 7. 个性化

- **主题** `⌘⇧L` 明暗切换（也可点标题栏 ◐）
- **字号** `⌘+` / `⌘-` / `⌘0` 复位；行宽在设置中调整
- **侧栏** `⌘\` 显示/隐藏
- **关于**：设置（`⌘,`）左侧导航「关于」——版本信息、GitHub 仓库/手册/更新日志入口、作者联系方式

## 9. 隐私

- 无遥测、无统计上报；除你配置的 AI 端点外不产生任何网络请求
- API Key 只存系统钥匙串；工作区索引（RAG）存本地 SQLite
- 文档永远是你磁盘上的纯文本，卸载 bmd 不影响任何数据
